# Zentrix | Software • Technology • Solutions

## Project Description
Zentrix delivers end-to-end technology solutions — from product ideation to deployment — for ambitious businesses. This project showcases the services and capabilities of Zentrix, focusing on a clean, modern, and responsive web presence.

## Features & Services
- **API Development & Integration:** Robust backend solutions and seamless integrations to connect your digital ecosystem.
- **E-Commerce Solutions:** Custom storefronts and payment gateways optimized for high conversion rates.
- **Tech Consulting & Strategy:** Aligning business goals with the right technology stack for sustainable scaling.
- **Founder-Led Quality:** Direct engagement with the engineer architecting your product.
- **Transparent Process:** Clear milestones, no surprise invoices, and weekly updates.
- **Built to Scale:** Code architected for growth, laying the foundation for a robust platform.

## Technologies Used

- CSS3 (with custom CSS variables for theming)
- JavaScript (for scroll animations, mobile menu toggle, and active navigation states)
- Responsive Design
